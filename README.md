# Simple Go Port Scanner
作为练习的简单Go端口扫描器.

**特点：**
 * Go routine 高性能并发,扫描本地0-65536端口大概5s
 * 通过Time.Ticker进行限频率扫描
 * 模拟Telnet接受端口协议信息

## Usage: 

```sh
# build (windows)
$ go build -o main.exe ./src 

# execute
$ ./main.exe -h

# or
$ go run ./src -h
```
